# PetMeCome
semi project "pet me come" 유기견 보호소 사이트 

# 프로젝트 설명
유기동물 인식 개선과 반려동물 입양 창구마련을 위한 유기동물 보호소 웹 사이트

# 주요 기능
1) 회원 : 회원가입, 로그인, 회원정보 조회 및 수정, 작성글 및 작성댓글 조회 , 아이디 및 비밀번호 찾기(이메일 전송), 암호화
2) 관리자 : 입양 게시글 등록
3) 게시판 : 입양 게시판, 신청 게시판, 후기 게시판 게시물 CRUD 및 댓글
